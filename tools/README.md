# Tools

This directory contains utilities for working with the repository.

## Contents

- [render_diagrams.sh](./render_diagrams.sh)  
  Generates diagram exports  

- [validate_xml.py](./validate_xml.py)  
  Validates draw.io XML structure  

---

## Purpose

These tools support:

- Diagram generation  
- Validation  
- Consistency across artifacts  

---

## Usage

Run tools from the repository root or within this directory.

---

## Key Idea

This is the **support layer** that enables reproducibility.
