# LLM Architecture System

A structured exploration of modern large language model architectures, including dense transformers, sparse mixture-of-experts systems, and hybrid compute patterns.

This repository organizes:
- canonical architecture archetypes
- a unified system model
- conceptual abstractions for reasoning about LLM design
- execution semantics for token flow and compute selection

## Repository Structure

- `docs/` — narrative documentation and system descriptions
- `diagrams/source/` — draw.io XML files (source of truth)
- `diagrams/rendered/` — exported PNG/JPG renders
- `diagrams/exports/` — optional PDF/SVG exports
- `models/` — conceptual frameworks and abstraction layers
- `examples/` — example interpretations and mappings
- `tools/` — helper scripts for rendering and validation

## Entry Points

1. `docs/overview.md`
2. `docs/archetypes.md`
3. `docs/unified_system.md`
4. `docs/execution_dag.md`

## Working with the Diagrams

1. Open https://app.diagrams.net
2. Import files from `diagrams/source/`
3. Edit or extend the diagram
4. Export PNG/JPG into `diagrams/rendered/`
5. Export PDF/SVG into `diagrams/exports/`
