# Core Concepts

This directory defines the foundational concepts for understanding large language model systems.

## Contents

- overview.md  
  - High-level framing of LLM systems  

- machine_blueprint.md  
  - Component-level breakdown of model structure  

- archetypes.md  
  - Canonical architectural patterns (Dense, MoE, Hybrid)  

- execution_dag.md  
  - Runtime execution model and token flow  

- unified_system.md  
  - Generalized architecture combining all archetypes  

- acronyms.md  
  - Standardized terminology and abbreviations

This directory answers:
